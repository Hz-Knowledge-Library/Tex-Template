Global Images Directory
======================

Place shared images here that are used across multiple chapters.

Examples:
- Logos
- Icons
- Common diagrams
- University seals
- Frequently used graphics

Supported formats: PDF, PNG, JPG, EPS

Usage in LaTeX:
\includegraphics[width=0.8\textwidth]{filename.png}

Note: No need to specify the full path, just the filename.
